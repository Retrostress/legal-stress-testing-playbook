# Legal Stress Testing Playbook

This repository contains practical templates and checklists for authorized network stress testing, load testing, and DDoS resilience validation.

The goal is simple: help teams test infrastructure they own or have explicit permission to test, without crossing into unauthorized activity.

## What this includes

- A legal stress testing scope template
- A DDoS resilience checklist
- A PCAP replay workflow
- An API load testing checklist

## Important

Only test systems you own or have explicit written permission to test.

Unauthorized stress testing, IP booter use, or traffic generation against third-party targets can be illegal and harmful.
